Cada carpeta cuenta con un código y un archivo JSON de prueba para ingresar información 
a la base de datos correspondiente, las bases de datos con las que se cuenta son:
1.- chat
2.- homework
3.- students register
4.- teachers register
Nota: para ingresar de forma correcta los programas se requiere utilizar Node y tener una cuenta de MongoDB
y remplazar sus datos en la variable "uri", por su url correspondiente en la base de datos.

Enlace a la base de datos de MongoDB :
mongodb+srv://<db_username>:<db_password>@cluster0.2b8il.mongodb.net/?retryWrites=true&w=majority&appName=Cluster0
